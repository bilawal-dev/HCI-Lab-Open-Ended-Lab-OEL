## Objective

Design an interactive prototype of a Final Year Project (FYP) idea in Figma and evaluate its usability using the **Cognitive Walkthrough** method.

## Project — AI Visibility & Brand Recommendation Optimization Platform

A SaaS platform for **Generative Engine Optimization (GEO)**. It monitors how a brand is mentioned across the major AI assistants (ChatGPT, Claude, Gemini, Perplexity), simulates buyer-persona queries, and aggregates the responses into an **AI Visibility Score** along with content recommendations that close visibility gaps.

## Prototype scope

Seven clickable screens with full navigation between them:

1. Login
2. Register
3. Onboarding — submit brand URL and start first scan
4. Dashboard — AI Visibility Score, per-platform breakdown, sentiment trend
5. Visibility Detail — drill-down per LLM, sample query response, competitor comparison
6. Content Recommendations — prioritised briefs with score-impact figures
7. Billing & Credits — plan overview and credit usage

Consistent color scheme (indigo / purple / pink gradient), Inter typography, and interactive buttons throughout.

## Cognitive Walkthrough Evaluation

Four representative user tasks evaluated step-by-step against the four CW questions:

- Will the user know what to do?
- Will the user notice the correct action?
- Will the user understand the system response?
- Will the user successfully complete the task?

Findings are grouped by severity (High / Medium / Low) with concrete improvement suggestions. Full evaluation in the submission document below.
